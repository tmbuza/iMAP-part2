A.4 How to reload RDS or RData into R environment

  • RDS format e.g. foo.rds
    • foo <- readRDS(“RDataRDS/foo.rds”)
  • RData format e.g. foo.RData
    • load(“RDataRDS/foo.RData”, verbose = TRUE)
  • List objects in RData
    • lsdata(“foo.RData”)
